00030 /*
00031  @mainpage A strict JSON parser and generator for Objective-C
00032  
00033  JSON (JavaScript Object Notation) is a lightweight data-interchange
00034  format. This framework provides two apis for parsing and generating
00035  JSON. One standard object-based and a higher level api consisting of
00036  categories added to existing Objective-C classes.
00037  
00038  This framework does its best to be as strict as possible, both in what it accepts and what it generates. For example, it does not support trailing commas in arrays or objects. Nor does it support embedded comments, or anything else not in the JSON specification. This is considered a feature. 
00039  
00040  @section Features
00041  
00042  @li Easy-to-use API.
00043  @li Strict parsing & generation.
00044  @li Stack of error available in case of failure so you can easily figure out what is wrong.
00045  @li Optional pretty-printing of JSON output.
00046  @li Optionally sorted dictionary keys in JSON output.
00047  @li Configurable recursion depth for parsing, for added security.
00048  
00049  
00050  @section Links
